# WSO2 Integrator Projects

A collection of integration projects built with WSO2 Integrator.

## Overview

This repository contains various integration solutions and examples using WSO2 Integrator.

## Getting Started

### Prerequisites
- WSO2 Integrator installed and configured
- Java Development Kit (JDK)

### Installation

Clone the repository:
```bash
git clone https://github.com/Sachindu-Nethmin/wso2-integrator-projects.git
cd wso2-integrator-projects
```
